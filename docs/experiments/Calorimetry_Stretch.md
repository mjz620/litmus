# Experiment 3 — Calorimetry Stretch

## Scope

Coffee-cup calorimetry with deterministic `q = mcΔT` plus optional heat-loss/calorimeter constant term.

## Core interactions

- Measure volumes/masses.
- Record initial temperatures.
- Mix solutions.
- Observe temperature curve.
- Compute heat transfer and enthalpy estimate.
- Submit analysis with sign conventions and sig figs.

## Skills

- `temperature_measurement`
- `heat_equation`
- `sign_convention`
- `calorimeter_constant`
- `sig_figs`

## Common mistakes

- Wrong sign for exothermic/endothermic.
- Uses volume instead of mass without density assumption.
- Ignores calorimeter heat capacity when configured.
- Reports unrealistic precision.

## Build rule

Do not start this until titration, teacher loop, demo mode, eval harness, and precipitation plugin are stable.
